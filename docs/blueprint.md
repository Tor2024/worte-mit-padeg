# **App Name**: Deutsche Worte mit Oleh

## Core Features:

- Word/Phrase Input: Allows users to input German words and phrases for learning. Checks for mandatory article for nouns.
- Automatic Analysis: Automatically classifies words (nouns, verbs, etc.) and determines verb conjugations and tenses. Then the appropriate data (like article and gender for nouns) is collected, saved and displayed.
- Usage Examples: Provides example sentences for words and phrases, highlighting important elements such as articles and verb forms. It uses AI as a tool to improve the usefulness of examples by including examples where the tool reasons if a given word in an example helps better demonstrate meaning in a given case, to ensure relevance and avoid hallucination of word definitions.
- Learning Systems: Offers various learning methods like flashcards and quizzes (article selection, verb form selection, sentence construction) to reinforce learning.
- Progressive Learning: Analyzes user progress and adapts the repetition of words and phrases for optimized learning.
- Intelligent Correction: Provides intelligent error checking with AI explanations and hints, including article verification for nouns and flexible assessment for other word types using meaning, variations and/or synonyms.
- AI-Powered Content Generation: Generates new sentences with learned words and suggests related words and phrases, personalizing and optimizing the learning process.

## Style Guidelines:

- Primary color: Light desaturated blue (#1E3A8A) to create a calm and focused learning environment.
- Background color: Very light blue (#F0F8FF) for a clean and unobtrusive backdrop.
- Accent color: Soft orange (#2E7D32) to highlight key elements and interactive components, providing a warm contrast.
- Font pairing: 'Inter' (sans-serif) for body text and 'Georgia' (serif) for headings, to combine clarity with a touch of elegance.
- Simple, intuitive icons for categories and learning modules to aid navigation and comprehension.
- Clean, spacious layout with clear visual hierarchy, emphasizing important elements like highlighted words and progress indicators.
- Subtle animations for feedback and transitions to enhance user engagement without being distracting.
